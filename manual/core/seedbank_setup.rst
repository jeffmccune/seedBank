seedbank_setup

Application Help

.. include:: help/seedbank_setup

Arguments
=========

The argument should be one of the available releases which can be listed by using the -l option

Examples
========

Download and install the needed syslinux files, get the Debian Squeeze amd64 netboot and integrate the 'non free firmware files' into it.

.. code-block:: none

    seedbank_setup -s
    seedbank_setup debian-squeeze-amd64
