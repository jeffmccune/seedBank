Application Help

.. include:: help/seedbank_daemon

Options
=======

Query options
-------------

-h Help, show the above help page

Daemon options
--------------

-d Run the seedBank daemon in daemon mode

Examples
========

Start the seedBank daemon (requires installation via the Debian package for the init script)

.. code-block:: none

    /etc/init.d/seedbank start
